# VSporte.Task

## VSporte.Task.Test
#### Description
Описание к задачам: https://gitlab.com/vsporte_public/vsporte-task

От разработчика:

Файлы для тестов находятся в каталоге Fingerprint. Файлы с эталонным ответом имеют префикс correct_.

Команды без игроков удаляются.

Имена и названия перед поиском дублей приводятся к единому формату, после удаления дублей формат возвращается в исходный вид.
